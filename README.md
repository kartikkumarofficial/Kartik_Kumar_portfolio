# 🌐 Kartik Kumar — Portfolio

A modern, responsive personal portfolio website built with **Dart** and **Flutter Web**, showcasing my projects, skills, and experience in a clean and interactive way.

## ✨ Features

- 🎨 **Modern UI** — Minimal, elegant design for an engaging browsing experience  
- 🌓 **Dark & Light Mode** — Seamlessly switch between themes for optimal viewing comfort  
- 📱 **Responsive Layout** — Optimized for desktop, tablet, and mobile devices  
- ⚡ **Fast & Smooth** — Powered by Flutter’s performant rendering engine  
- 🖼️ **Custom Branding** — Personalized favicon, color theme, and animations  
- 📂 **Projects Showcase** — Highlights of my work with live previews & source links  
- 📜 **Resume/CV Section** — Downloadable PDF of my professional background  
- 📬 **Contact Form** — Easy way for visitors to connect with me

## 🛠️ Tech Stack

- **Language:** Dart  
- **Framework:** Flutter (Web)  
- **Hosting:** Vercel  
- **State Management:** Provider (or GetX/Riverpod - specify if used)
- **Assets:** Custom icons, images, and animations
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/yourusername/your-portfolio.git
cd your-portfolio
````

### 2️⃣ Install Dependencies

```bash
flutter pub get
```

### 3️⃣ Run the Project Locally

```bash
flutter run -d chrome
```

### 4️⃣ Build for Web

```bash
flutter build web
```

The compiled files will be available in the `build/web` directory.

## 📦 Deployment

This portfolio is hosted on **Vercel**.
After building for the web, deploy using:

```bash
vercel --prod
```
